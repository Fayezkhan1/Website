# VNIT Logo Setup Instructions

## How to Add the VNIT Logo

1. **Save the logo image** you shared as `vnit-logo.png`
2. **Place it in** the `frontend/public/` folder
3. The logo should be the circular VNIT emblem with:
   - "VISVESVARAYA NATIONAL INSTITUTE OF TECHNOLOGY" text around the border
   - "NAGPUR" at the bottom
   - The motto "योग: कर्मसु कौशलम्" (Yoga: Karmasu Kaushalam) in the banner

## File Location
```
frontend/
  └── public/
      └── vnit-logo.png  ← Place the logo here
```

## Recommended Specifications
- **Format:** PNG with transparent background (preferred) or white background
- **Size:** At least 500x500 pixels for best quality
- **File name:** Exactly `vnit-logo.png` (lowercase)

## Alternative Method
If you have the logo file on your computer:
1. Navigate to the `frontend/public/` folder in your file explorer
2. Copy/paste or drag-drop the logo file there
3. Rename it to `vnit-logo.png` if needed
4. Refresh your browser to see the logo appear

## Current Status
The website is already configured to display the logo. Once you add the file, it will automatically appear in the navigation bar.
