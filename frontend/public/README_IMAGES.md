# Image Setup Instructions

Please add the following images to the `frontend/public` folder:

1. **vnit-logo.png** - VNIT college logo (recommended size: 200x200px)
2. **hostel1.jpg** - Main hostel building exterior
3. **hostel2.jpg** - Student room interior
4. **hostel3.jpg** - Common area/recreation room
5. **hostel4.jpg** - Dining hall

You can download the official VNIT logo from the VNIT website or use placeholder images for now.

For placeholder images, you can use services like:
- https://placeholder.com/

- Or take actual photos of your hostel facilities
